# video-to-steps

Local web app: paste a YouTube instructional video URL, get back a
sequential list of written instructions, each illustrated with the
most relevant frame(s) from the video.

**Status: design only.** See [PLAN.md](PLAN.md) for the full project
plan. An implementing agent should read PLAN.md end-to-end before
writing any code; every architectural decision is recorded there.

Coding work for this project is intended to be done by Claude Code
(or a comparable agent) from the plan, end-to-end. Disclosure matches
the qwen-studio sibling project.
